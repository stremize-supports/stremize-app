# Stremize

Official site: https://stremize.com

*Current we only support Android mobile / TV*

- Easy to use, support 4K
- You must add addons to play movie, we use your provider for get original link
- Stremize keeps track of your account configuration. Log in and continue watching without having to reconfigure addons on a new device.
